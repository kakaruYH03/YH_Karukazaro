# YH_UI

<br>

#### 유니티에서 사용가능한 게임 UI

- 2019/11/13 Ver

---
###### 사용 가능한 기능들

    1. 하단의 더 보기탭으로 사용가능한 탭 변환 기능
    2. 탭 클릭시 다른 화면으로 이동기능 
    3. 퀘스트 창에서 퀘스트를 클릭시 상세정보 확인 기능
    4. 상단에서 현재 장소와 체력 표시

---

#### 구현될 예정인 기능

    1. 다른 탭 화면으로 이동시 위에서 아래로 애니메이션으로 내려오기
